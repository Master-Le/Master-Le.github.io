1-4.流式文件

程序打开一个文件是需要消耗资源的，流式写入可以减少打开关闭文件的次数

流式写入方式适用于大文件写入或者频繁写入的场景，writeFile适用于写入不频繁的场景

1-5.写入文件场景

以下场景用到文件写入

  • 下载文件
  • 安装软件
  • 保存程序日志、如git
  • 编辑器保存文件
  • 视频录制

需要持久化保存数据的时候，就要想到用文件写入

四、path模块

API 说明
path.resolve 拼接规范的绝对路径(常用)
path.sep 获取操作系统的路径分隔符
path.parse 解析路径并返回对象
path.basename 获取路径的基础名称
path.dirname 获取路径的目录名
path.extname 获得路径的拓展名

五、http模块

1.创建http

**注意事项

ctrl + c 停止服务

当服务启动后，更新代码必须重启服务才能生效

响应内容中文乱码的解决方法

Response.setHeader(‘content-type’,’text/html;charset=utf-8’)

端口号被占用

Error: listen EADDRINUSE: address already in use :::9000

关闭当前正在运行监听端口的访问

修改其他端口号

HTTP协议默认端口为80，HTTP服务开发常用端口有3000，8080，8090，9000等

如果端口被其他程序占用，可以用资源监视器找到占用端口的程序，然后使用任务管理器关闭对应的程序

7.网页资源加载基本过程

前端发送请求获取html文件并解析，如果有css文件就继续发请求获取css文件并解析，然后如果有静态资源如图片类继续发请求获取并渲染，最后获取js文件并解析

8.静态资源和动态资源

静态资源：内容长时间不发生改变的资源，如图片，视频，css文件，js文件，html文件等

动态资源：内容经常更新的资源，例如百度首页，网易首页等

8.1网站根目录或静态资源目录

HTTP服务在哪个文件夹中寻找静态资源，那个文件夹就是静态资源目录也叫网站根目录
